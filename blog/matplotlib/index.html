简介¶

matplotlib可能是2D绘图最常用的Python包，它提供了快速的方法对数据可视化，这篇文章介绍了最常用的例子。

pyplot¶

pyplot为matplotlib面向对象的绘图库提供了一个方便的接口。它是在Matlab(TM)的基础上建立的。因此，pyplot中的大多数绘图命令都具有Matlab(TM)类似的参数。重要的命令通过交互式示例进行解释。

简单绘图¶

我们准备在同一个plot上画出sin函数和cos函数，从默认设置出发，一步一步进行完善。

Figures,Subplots,Axes and Ticks¶

在开始之前，强烈建议先看下知乎的这篇文章python matplotlib中axes与axis的区别?

$\bf{〝}$一个完整的画图过程可以使用两套工具分别来实现：

. 函数式编程思想：调用pyplot中的 plot() 函数和 xlim() 函数。

. 对象式编程思想：使用生成的Subplot对象的两种方法 .plot 和 .set_xlim方法。

$\bf{〞}$ -----知乎$\bf{·}$禹洋

到此为止，我们大部分使用的是第一种方式，这对快速画图是很方便的。事实上，我们可以使用figure,subplot,axes显式的施加更多控制。 A figure in matplotlib means the whole window in the user interface. Within this figure there can be subplots. While subplot positions the plots in a regular grid, axes allows free placement within the figure. Both can be useful depending on your intention. 我们已经以一种隐式的方式调用过figure和subplot了，当我们调用plot时，matplotlib调用gca()来获取当前axes，gca反过来调用gcf()来获取当前figrue.如果没有figure，则调用figure()生成一个，严格地说，生成一个subplot(111)。

Figures¶

图是GUI中标题为“figure #”的窗口。图的编号是从1开始的，而不是普通的Python方法从0开始。这显然是跟随了MATLAB风格的。下面是一些figure的参数。

Argument Default Description
num 1 number of figure
figsize figure.figsize figure size in in inches (width, height)
dpi figure.dpi resolution in dots per inch
facecolor figure.facecolor color of the drawing background
edgecolor figure.edgecolor color of edge around the drawing background
frameon True draw figure frame or not

大多数情况下使用默认值。只有$num$是经常变化的。

当你使用GUI时，您可以通过单击右上角的×来关闭figure。你还可以通过调用close以编程方式关闭它。根据参数的不同，它关闭(1)当前图(无参数)、(2)特定的图(图号或图实例作为参数)或(3)所有图(all作为参数)。

与其他对象一样，可以使用$set\_something$方法设置figure属性

Ticks¶

ticks的格式有时也很重要。Matplotlib提供了一个完全可配置的ticks系统。有一些ticks定位器来指定ticks应该出现在何处，以及ticks格式化器来给出所需的ticks外观。major ticks和minor ticks可以彼此独立定位和格式化。默认情况下，不显示minor ticks，也就是说，只有一个空列表，因为它是空定位器(见下面)。

Class Description
NullLocator

No ticks.
IndexLocator

Place a tick on every multiple of some base number of points plotted.
FixedLocator

Tick locations are fixed.
LinearLocator

Determine the tick locations.
MultipleLocator

Set a tick on every integer that is multiple of some base.
AutoLocator

Select no more than n intervals at nice locations.
LogLocator

Determine the tick locations for log axes.

所有这些定位器都派生自基类matplotlib.ticker.Locator。你可以从它派生出你自己的定位器。将日期作为ticks进行处理尤其棘手。因此，matplotlib在matplotlib.date中提供了特殊的定位器。
