Scikit-Learn的数据表示¶

数据表¶

Seaborn是基于matplotlib的Python可视化库,它提供了一个高级界面来绘制有吸引力的统计图形，iris是鸢尾花数据集。

In [1]:
import seaborn as sns
iris = sns.load_dataset('iris') #type(iris) -> pandas.core.frame.DataFrame
iris.head()
Out[1]:
sepal_length sepal_width petal_length petal_width species
0 5.1 3.5 1.4 0.2 setosa
1 4.9 3.0 1.4 0.2 setosa
2 4.7 3.2 1.3 0.2 setosa
3 4.6 3.1 1.5 0.2 setosa

每一行表示一个样本（sample），行数表示样本数，记为n_samples；同样，每一列表示一个特征(feature)，列数记为n_features 。

特征矩阵¶

上面的表格称为特征矩阵（features matrix），通常被简记为变量$X$，维度为[n_samples,n_features],可以使用NumPy的数组、Pandas的DataFrame来表示，不过Scikit-Learn也支持SciPy的稀疏表示。

目标数组¶

目标数组一般是长度为n_samples一维数组，通常用NumPy数组或Pandas的Series表示。目标数组的特征通常是我们希望从数据中预测的量化结果。

Scikit-Learn的评估器API¶

.统一性
所有对象使用共同接口连接一组方法和统一的文档。
.内省
所有参数值都是公共属性。
.限制对象层级
只有算法可以使用Python类表示。数据集都用标准数据类型（NumPy数组、Pandas、DataFrame、SciPy稀疏矩阵）表示，参数名称用标准的Python字符串。
.函数组合
许多机器学习任务都可以用一串基本算法实现，Scikit-Learn尽力支持这种可能。
.明智的默认值
当模型需要用户设置参数时，Scikit-Learn预先定义适当的默认值。

API基础知识¶

（1）通过从Scikit-Learn中导入适当的评估器类，选择模型类。

（2）用合适的数值对模型类实例化，配置模型超参数（hyperparameter）。

（3）整理数据，通过前面介绍的方法获取特征矩阵和目标数组。

（4）调用模型实例的fit()方法对数据进行拟合。

（5）对新数据应用模型：

    .在有监督学习模型中，通常使用predict()方法预测新数据的标签；

    .在无监督学习模型中，通常使用transform()或predict()方法转换或推断数据的性质。

有监督：鸢尾花分类示例¶

这里使用非常简单的高斯朴素贝叶斯（Gaussian naive Bayes）方法，“朴素贝叶斯方法实际上学习到生成数据的机制，所以属于生成模型。条件独立假设等于是说用于分类的特征在类确定的条件下都是条件独立的。这一假设使朴素贝叶斯法变得简单，但有时会牺牲一定的分类准确率。”（李航《统计学习方法》），朴素贝叶斯的参数估计包括极大似然估计和贝叶斯估计，贝叶斯估计解决极大似然估计可能会出现所要估计的概率值为0的情况。

因为高斯朴素贝叶斯方法速度很快，不需要选择超参数，所以通常很适合作为初步分类手段。

In [17]:

无监督：鸢尾花数据降维示例¶

降维的任务是要找到一个可以保留数据本质特征的低维矩阵来表示高维数据。主成分分析（Principal Component Analysis，PCA）方法是一种快速线性降维技术。

In [19]:
from sklearn.decomposition import PCA
model = PCA(n_components=2)
model.fit(X_iris)
X_2D = model.transform(X_iris)
In [26]:
iris['PCA1'] = X_2D[:,0]
iris['PCA2'] = X_2D[:,1]
sns.lmplot('PCA1','PCA2',hue='species',data=iris,fit_reg=False)
Out[26]:
<seaborn.axisgrid.FacetGrid at 0x170ff3c1240>

无监督：鸢尾花数据聚类¶

聚类算法是要对没有任何标签的数据集进行分组。这里使用高斯混合模型（Gaussian mixture model，GMM）,高斯混合模型是对高斯模型进行简单的扩展，GMM使用多个高斯分布的组合来刻画数据分布。

In [27]:
from sklearn.mixture import GaussianMixture
In [28]:
model = GaussianMixture(n_components=3,covariance_type='full')
model.fit(X_iris)
y_gmm = model.predict(X_iris)
In [30]:
iris['cluster'] = y_gmm
sns.lmplot('PCA1','PCA2',data=iris,hue='species',col='cluster',fit_reg=False)
Out[30]:
<seaborn.axisgrid.FacetGrid at 0x17082f8f978>

蓝色点(setosa)从cluster0中被完美分离，绿色点（versicolor）和红色点（virginical）还有混淆。
